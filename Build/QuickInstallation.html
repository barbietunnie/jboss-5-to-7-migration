Quick Start Guide

Installation:

  1. Unzip the downloaded file into a folder, for example: c:\esphere.
  2. Install MySQL version 5 on your server if you haven't done so (If you are installing your MySQL on another computer, make sure to allow the root user to login from a remote host).
  3. Open "customize.properties" file under c:\esphere\build and modify mysql.host and mysql.port properties to point to your MySQL instance.
  4. Open a DOS window and cd to c:\esphere\build.
  5. Type "ant deploy" and press Enter key.
  6. Type "ant create_db" and press Enter key, type your MySQL server's root password when asked.
  7. cd to c:\esphere\build\jboss-4.2.1.GA\bin and type "run" and press Enter to start the server, to stop the server, press Ctrl-C, and answer "Y" when asked.
    (For security, the Emailsphere System Management Console can only be accessed locally. If you want to access Emailsphere System Management Console from other machines, use this command to start the server: run -b 0.0.0.0)

Configuration:

  1. Open a browser and point the browser to: http://localhost:8080/MsgUI/.
  2. Login as administrator using admin/admin as user id and password.
  3. Click on "Configure Site Profiles".
  4. Click on "System".
  5. Make changes accordingly and save the changes.
  6. Back to main page and click on "Configure POP/IMAP Accounts".
  7. Add your POP3 or IMAP accounts and save the changes.
  8. Back to main page and click on "Configure SMTP Servers".
  9. Add your SMTP servers and save the changes.

Restart server:

  1. Go back to the DOS window where the server is running.
  2. Press and hold Control button and then press button "C".
  3. Type "Y" when asked.
  4. Type "run" and press Enter key to restart the server.

Apache James Server:

Eamilsphere all-in-one package comes with a pre-configured POP/SMTP server. It is located under c:\esphere\james-2.3.1.
  • To start James server, open a DOS window, cd to c:\esphere\james-2.3.1\bin, type "run" and press Enter.
  • To stop the James server, go to the DOS window where James server was started, and press Ctrl-C, type "Y" when asked (ignore the error messages on the DOS screen).

System Requirements:

  • Operating Systems: Windows 2000, Windows XP, Windows Vista
  • Processor Speed: Intel or AMD 1.6GHz or higher
  • System Memory: 1GB or more
  • Free Disk Space: 10GB or more
